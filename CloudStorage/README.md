# Проект CloudStorage

Проект CloudStorage представляет собой простую систему для хранения файлов в облаке. Этот проект создан для демонстрации возможностей облачных хранилищ и применения базовых концепций облачных технологий.
## Состоит  из 3 проектов:
* **CloudStorageWebAPI** - 
* **CloudStorageWeb** - 
* **CloudStorageClass** -
* **CloudStorageWPF** - 

## Функциональности

- **Регистрация и аутентификация пользователей**: Пользователи могут зарегистрироваться, создав учетную запись, и войти в систему, используя свои учетные данные.
- **Загрузка файлов**: Аутентифицированные пользователи могут загружать файлы на сервер.
- **Хранение файлов в облаке**: Загруженные файлы хранятся на сервере для последующего доступа.
- **Удаление файлов**: Пользователи могут удалять файлы из своего облачного хранилища.
- **Просмотр списка файлов**: Пользователи могут просматривать список всех файлов, загруженных в их облачное хранилище.
